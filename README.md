# README

## Usage

- `python3/py start.py [-OPTION]`
- Options: [-h | --help | -H] [-o | --options | -O] [-v | --version | -V]

## Configuration File

1. Make a file in the project's root directory called `config.ini`.
2. Add the attributes below as a starting point.

```ini
[DEFAULT]
browser = dd
editor = micro

[DAILY_NOTES]
enabled = true
folder = DIARY
```
